# QA Automation Demo Repository

This repository contains a practical demonstration of my QA Automation skills, covering both **frontend** and **API testing**.

## Frontend Automation – Playwright (Python)

Automated end-to-end tests for https://www.saucedemo.com/ using:

- Python
- Playwright
- Page Object Model (POM)
- Assertions and validation strategies
- Test structure following automation best practices

---

## API Testing – SWAPI (Postman)

API test suite built against https://swapi.dev/api/ using Postman, applying best practices such as:

- Environment variables
- Pre-request and post-response scripts
- Data persistence between requests
- Type and schema validations
- Pagination testing
- Negative scenarios
- Cross-endpoint validations
